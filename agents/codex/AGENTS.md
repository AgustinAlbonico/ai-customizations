# AGENTS - Codex

Base de comportamiento para usar customizaciones de este repositorio con Codex.

## Objetivo

- Reutilizar skills y comandos de manera consistente.

## Reglas

- Preferir skills del repo antes de crear prompts largos ad-hoc.
