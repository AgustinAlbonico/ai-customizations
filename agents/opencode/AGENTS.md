# AGENTS - OpenCode

Estas instrucciones son una base reusable para proyectos donde instales este pack.

## Objetivo

- Mantener consistencia entre skills, comandos y hooks.
- Priorizar cambios pequenos, verificables y seguros.
- Ejecutar verificaciones antes de cerrar una tarea.

## Reglas operativas

- Usar skills del repo para flujos repetibles.
- Registrar decisiones tecnicas en mensajes finales de cada tarea.
- Evitar cambios destructivos en git salvo instruccion explicita.

## Verificacion minima

- Revisar estado de git.
- Validar que la skill/comando aplicado corresponda al objetivo.
- Confirmar que no quedan pasos criticos pendientes.
