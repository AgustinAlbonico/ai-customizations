# /tauri-migrate

Ejecutar flujo completo de migracion Web -> Desktop con Tauri para stack React + NestJS + Postgres en LAN.

## Flujo

1. Planificacion tecnica
2. Implementacion
3. Verificacion funcional y de logs
4. Preparacion de release

## Regla LAN

- Backend local en `127.0.0.1`
- DB remota por IP fija en `database.host`
